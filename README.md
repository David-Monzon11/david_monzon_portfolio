## Setup Instructions

### 1. Add Your Profile Image

Place your profile photo in the root directory and name it `profile.jpg`. The image should be:
- Square or portrait orientation (recommended: 320x320px or larger)
- High quality (JPG or PNG format)
- Professional headshot

If your image has a different name or format, update the `src` attribute in `index.html`:
```html
<img src="your-image-name.jpg" alt="David D. Monzon" class="profile-image" />
```

### 2. Update Social Media Links

Make sure your social media links in `index.html` are correct:
